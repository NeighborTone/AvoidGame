# 概要

+ 自分で作ったクラスを使ったサンプルゲーム
+ 勉強の息抜きで制作
+ 5日間でどこまでできるかの実力試し
+ フリーゲーム夢現に投稿しました https://freegame-mugen.jp/action/game_6887.html
## ゲーム内容

+ 障害物をひたすらよけるゲームです

+ 障害物に当たるか地面に落ちるとゲームオーバーです。

## 操作方法

+ Zキーを押している間は上昇

+ 離している間は下降

+ ←→キーで移動	

+ タイトルでRキーを押しながらEキーを押すとハイスコアがリセットされます

+ Escキーで終了できます
